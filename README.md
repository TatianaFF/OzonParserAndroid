# Ozon Price Tracking

Приложение для мониторинга цен на товары в маркетплейсе Ozon. Позволяет пользователям отслеживать изменения стоимости интересующих товаров и просматривать историю изменений.

## Основные возможности
*   **Добавление товаров**: Возможность добавить товар для отслеживания (по ссылке).
*   **Список товаров**: Удобный просмотр всех отслеживаемых позиций с текущими ценами.
*   **Детализация**: Просмотр подробной информации о товаре и истории изменения цены.
*   **Фоновое обновление**: Автоматическая проверка цен в фоновом режиме с использованием WorkManager.
*   **Уведомления**: (В разработке) Оповещение пользователя при снижении цены.

## Стек технологий
*   **Язык**: Kotlin
*   **UI**: Jetpack Compose (Material 3)
*   **Архитектура**: MVVM, Clean Architecture
*   **DI**: Hilt (Dagger)
*   **База данных**: Room (для хранения товаров и истории цен)
*   **Фоновые задачи**: WorkManager
*   **Сеть/Парсинг**: (Kotlin Serialization для обработки данных)
*   **Логирование**: Timber
*   **Аналитика и отчеты об ошибках**: Firebase (Analytics, Crashlytics)
*   **Загрузка изображений**: Coil

## Структура проекта
Проект организован по принципу Feature-based структуры:

*   `core/` — общие компоненты приложения:
    *   `database/` — локальное хранилище данных (Room).
    *   `logging/` — конфигурация логирования.
    *   `products/` — общие компоненты (репозитории, модели данных, воркер, парсер).
    *   `theme/` — общие стили и темы Jetpack Compose.
*   `feature/` — функциональные экраны:
    *   `home/` — главный экран со списком товаров.
    *   `details/` — экран деталей товара.
    *   `createProduct/` — экран добавления нового товара.

## Установка и запуск
1. Клонируйте репозиторий.
2. Откройте проект в Android Studio (Ladybug или новее).
3. Добавьте свой файл `google-services.json` в папку `app/` для работы Firebase (или отключите плагин в `build.gradle`, если аналитика не требуется).
4. Соберите и запустите приложение на эмуляторе или реальном устройстве (minSdk 24).
